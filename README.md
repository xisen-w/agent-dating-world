# Agent Dating World

Aicoo Dating World is a frontend-first prototype for a playful agent society simulation. Users configure a personal COO/avatar, send it into a small dating world, and watch agent-to-agent interactions emerge as movement, dialogue, relationship state, and event logs.

## Local Development

```bash
pnpm install
pnpm dev
```

## Current Scope

- Vite + React + TypeScript frontend
- Mock simulation loop
- Avatar configuration panel
- Live cafe map with animated agents
- Event ticker, selected-agent state, relationship meters

The backend and Aicoo API integration are intentionally left for the next phase.
